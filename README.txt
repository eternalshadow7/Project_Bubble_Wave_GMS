Project: Bubble Wave
Made by Janelynn Camingue
Date started: 1/22/17
Date finished: 1/22/17
Time it took: 3-4 hours
Using Game Maker Studio free version on Steam

Controls: left, right, up, and down arrow keys
Download the .exe file and play.

Objective: collect all the bubbles and avoid the red triangles and hitting the darker blue sides. Reach the green
box at the end to finish.
